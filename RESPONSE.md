# Voice Command Update: Hindi Only

I have updated the voice command system to strictly understand only **Hindi** (`hi-IN`) as requested.

## Summary of Changes

1.  **Strict Hindi Input**:
    *   The microphone will now only listen for Hindi input.
    *   English input is disabled (`en-US` removed from configuration).

2.  **Wake Word**:
    *   You can now say only **"Radhe Radhe"** (matches "राधे राधे" in Devanagari script).

3.  **Expanded Hindi Support**:
    *   Added support for common Hindi/Hinglish trading terms.
    *   Examples: "bhav batao" (price), "dikhao" (show), "analysis karo", "top 5 stock".
    *   Fixed handling of Devanagari numerals (e.g., "टॉप 5").

## How to Use

1.  Click the microphone button.
2.  Say **"Radhe Radhe"**.
3.  Speak your command (e.g., "Reliance ka bhav batao").

The system is now optimized for Hindi interaction.
